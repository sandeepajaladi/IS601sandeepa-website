Here are the Linux commands

  1. CD = Change Directory
  2. Mkdir NAME = Make Directory
  3. Ls = list files
  4. Vi FILENAME
  5. I = insert mode
  6. Escape = get out of insert mode
  7. Shift key + colon = command prompt for vi
  8. W = command to write
  9. Q = command to quit
  10. WQ = Write and Quit
    11. :w:

Git Commands

  1. Git init = Create the repository
  2. Git Add filename = add the file(s) to git in preparation for a commit
  3. Git commit -m “message”= do the commit to save changes
  4. Git checkout -b branch name = make a new branch and checkout
  5. Git checkout branch name = checkout branch
  6. Git merge BRANCHNAME = merge the branch Note: You have to be on the branch that you want to merge to
